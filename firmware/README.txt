
Arduino_WS2812
--------------
You'll need to get the Adafruit NeoPixel library

  https://github.com/adafruit/Adafruit_NeoPixel


AVR_WS2812
----------
This is from the light_WS2812 library: https://github.com/cpldcpu/light_ws2812

It's currently set up for ATtiny25, but you can change the ws2812_config.h

cd into the directory and type make to build the hex files. Then flash them to your AVR
